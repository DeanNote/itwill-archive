html5 추가 폼 요소들

1. input type="time" - 시간(오전/오후 시 분)을 지정하는 폼요소

2. input type="date" - 시간(연월일)을 지정하는 폼요소


(min23/10/05~max23/12/31)

1+2 input type="datetime-local" - 시간(연월일 오전/오후 시분)을 지정하는 폼 요소

3. input type="week" - 시간(주)을 지정하는 폼요소

4. input type="month" - 시간(월)을 지정하는 폼요소

5. input type="tel" - 연락처(전화번호)를 지정하는 폼 요소

PC브라우저에선 그냥 textbox
모바일 브라우저에서 지원하는 경우 가상키패드가 숫자형식으로 바뀌고 전화걸기 가능.

6. input type="email" - 연락처(이메일)를 지정하는 폼 요소

PC브라우저가 알맞은 형식(@유무,@이후 완성)인지 확인해줌.
모바일브라우저에서 지원하는 경우 가상키패드가 @, .com 등이 나타남

7. input type="URL" 연락처(웹 주소)를 지정하는 폼 요소

PC브라우저가 알맞은 형식(유무)인지 확인해줌.
모바일브라우저에서 지원하는 경우 가상키패드가 , .com 등이 나타남

8. input type="color" - 색을 지정하는 폼 요소

9. input type="number" - 숫자값을 지정하는 폼요소

min="n"과 max="n"로 최소/최대값을, step="n"로 증감간격을 지정 가능


(min="0")
(max="20")
(step="10)

10. input type="range" - 범위내의 값을 지정하는 폼요소




html 추가된 속성들

1. autofocus(autofocus="autofocus") -해당요소가 자동 포커싱 됨

2. autocomplete="on/off" - 자동 완성기능을 켜거나 끔

3. required(required="required") - 필수입력

4. placeholder - 입력예시를 보여주거나 설명 등을 추가


5. readonly(readonly="readonly") - 읽기전용모드

6. disabled(disabled="disabled") - 사용불가모드
